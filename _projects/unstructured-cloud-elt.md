`Unstructured Cloud ELT` is an end-to-end pipeline for converting unstructured support and chat data into analytics-ready signals.

## Highlights

- Built an automated ELT pipeline to ingest chat screenshots into BigQuery using Vision AI for text extraction.
- Added sentiment and intent classification using BigQuery ML for downstream analytics and response automation.
- Connected outputs to dashboards in Looker Studio for operational monitoring.

## Tech Stack

BigQuery, Gemini API, Looker Studio, Vision AI, Python.
