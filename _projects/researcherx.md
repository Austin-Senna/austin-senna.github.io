`ResearcherX` is an AI-powered IDE for academic writing with real-time logical contradiction linting.

## Highlights

- Built a dual-routing pipeline with FastAPI and `litellm` to optimize latency and API cost between local and frontier models.
- Engineered asynchronous hybrid retrieval across Neo4j and LanceDB.
- Implemented node-level provenance and Cypher-based garbage collection to preserve graph integrity and reduce hallucinations.

## Tech Stack

Python (FastAPI), TypeScript (Next.js), Neo4j (Cypher), LanceDB, litellm, ProseMirror.
